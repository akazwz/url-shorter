# 短链生成

## 简单的短链生成器 用 next.js 和 redis 构建

## 在线 demo [url shorter](https://zlz.pw/)

## 用到的技术或者库:

- next.js (全栈框架)
- vercel (serverless 部署)
- redis (kv存储，储存 短链标识符 和 长链接)
- upstash (serverless redis 数据库)
- ioredis (redis 库)
- nanoid (生成短链标识符)
- is-url (判断是否是合法的链接)
- chakra-ui (前端 UI 框架)
- @icon-park/react (图标)
